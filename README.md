# cars

Exercise to start a rest api with java and "add" a car with the post route.

## Routes

```yml
POST /api/cars
    - Route to add a car.
    - headers: {}
    - body: {
	            modelo: "Fiat Argo 1.3",
	            fabricante: "Fiat",
	            dataFabricacao: "10/02/2006",
	            valor: 10.000,
	            anoModelo: 2023,
            }
```
